#北漂生活#培训见闻

十一假期的时候，中国交通运输堪比春运，我好不容易网上买了一张回家的硬座。
这趟车人好多，过道上全是人，好不容易来到座位。想想在过道上都挤满人的车
箱里坐八九个小时，想想我为了回家也是蛮拼的。

我的座位是挨着过道的，我对面坐着一个西装革履的小伙，白衬衫加黑长裤，目
光坚定，声音洪亮，给个代号A，A正在和他同座的B讨论问题，B也是相同的装扮
但是B显得很不自信。座子上摆着一台联想的Y500电脑。坐在我旁边是一个20多
岁的小伙，穿着T恤，给我一种同类的感觉，代号C吧。

A和B在笔记本上讨论着spring啊 HTML啊这些问题，我开始以为他们是哪个大
学的计算机的学生。坐车闲来无聊，我和C就一直听A和B的对话。

A：实习班一位学长毕业就30K，我这样水平的，简历“包装下”拿个15K小意思
B：还是你们实习班的厉害，我还的再培训几年呢。
他们的对话真是让我感到吃惊。一个专科都没有毕业，直接去培训机构培训两年的
新人，上来就敢要15K的薪水，我对这个IT培训行业也是无语了。

IT培训已经变味道了，你培训班教授IT启蒙知识没有问题，但是你公然教授简历
造假就是道德有问题了。

A继续说，实在找不到工作可以找导师推荐，xxx就是找导师入职的。。。

听到这句话，突然想起一个事，我加入过一个JAVA招聘群，里面有培训机构导师
推荐工作的，试用期不满就被开除了。。。真是各种套路啊。这个套路深的IT培
训机构良心真是坏了，为了提高就业率各种套路都使出来，坑了好多人。

这会出来一个路人甲，他跟AB说，你们也是做IT的么，我是做学习运维的，不用
写代码毕业就10K。。。。

好吧，这个路人甲初中刚毕业直接被骗到培训机构了，这些培训机构真是很没有
下线。。。

我和C一直看着AB还有路人甲在谈论，感觉他们中毒太深，培训机构套路太多。A看
着我说，哥们，你做什么的？我说，我们是同行，就是做JAVA开发的。A来了兴趣，
意思是怎么样我的能力还行吧，我也不忍心打击那个input框都无法对齐的页面，我
说，我工作3年也没有达到你的工资薪水，你们最好多看看招聘信息，多了解下市场
行情。不要被培训机构一家之言给蒙蔽了。我看着C说，这哥们应该也是IT的，你让
他说说看法，C果然是做IT的，做前端开发的，C说 他也达不到这么高的薪水，也委
婉表达了让他们好好看看IT行情，需要自己多多学习。

可能是受到了我们的触动，A转移了话题，A的目光中多了些思考和忧虑。。。

俗话说师傅领进门，修行靠个人。培训机构培训IT知识技能没有错，但是通过各种不
光彩的手段，提升就业率，骗取培训生的学费，就是不对的了。可怜这些被骗的学生。

诚者天之道也，思诚者人之道也--孟子

未完待续。。。
